# nema — React HTML-Gerüst

Dieses Gerüst ist ein Vite/React-Projekt für die Webseite **nema**. Es bildet das erste minimalistische Designkonzept als HTML/JSX-Struktur ab.

## Enthalten

- `index.html` als Vite-Einstieg
- `src/App.jsx` mit allen Seitenbereichen als React-Markup
- `src/components/Header.jsx` und `SectionHeading.jsx`
- `src/data/content.js` mit Mock-Daten für Werke, Künstler und Favoriten-History
- `src/styles.css` mit ruhigem, responsivem Galerie-Design
- `src/assets/img/nema-logo.png` mit dem bereitgestellten Logo

## Starten

```bash
npm install
npm run dev
```

Danach die lokale URL aus dem Terminal im Browser öffnen, meistens:

```bash
http://localhost:5173
```

## Weiterentwicklung

Die aktuellen Bereiche sind als Single-Page-Prototyp mit Anker-Navigation umgesetzt. Für ein echtes mehrseitiges React-Projekt kannst du später React Router ergänzen und die Sektionen in eigene Seiten auslagern.
